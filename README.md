# 🚀 DreamRoot – Stream & Career Guidance App

Discover your ideal stream with AI-powered quiz & career guidance.

---

## 📌 About the Project

DreamRoot is an Android application designed to help students choose the right academic stream (**Science, Commerce, Arts, Maths**) based on their interests.

It uses an interactive quiz system and AI-based logic to suggest suitable career paths along with curated learning resources.

---

## ✨ Features

The app includes multiple features such as Stream Finder, Interactive Quiz System, Performance Analysis, Career Suggestions, Learning Resources, and Language Support.

Stream Finder helps users identify the most suitable academic stream based on their interests.

The Interactive Quiz System provides MCQ-based questions with real-time progress tracking and bilingual support (English + Hindi).

Performance Analysis shows detailed results including correct answers, wrong answers, and accuracy percentage.

Career Suggestions recommend suitable career paths like Doctor, Pharmacist, Researcher, Biotechnologist, and Astronomer.

Learning Resources provide access to platforms like MIT OpenCourseWare, Coursera, Khan Academy, NASA, and National Geographic.

Language Support allows users to switch between English and Hindi for better accessibility.

## 🛠️ Tech Stack

Language used in this project is Java.

UI is built using XML and Material Design components.

The project is developed in Android Studio.

Charts are implemented using MPAndroidChart library.

AI features are powered using Gemini API.
